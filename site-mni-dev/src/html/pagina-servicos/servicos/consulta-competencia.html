Tribunal de Justiça do Estado de São Paulo

SERVIÇOS DISPONÍVEIS DO MNI 3.0. consultarCompetencias

Descrição:
Permite obter a listagem de todas as competências de uma determinada localidade judicial, informando como parâmetro o código da localidade que se deseja consultar.

Parâmetros (*campos opcionais*):
codigoLocalidade Código da localidade.


Se o valor da competência não existir o sistema não retorna erro, mas não retorna nenhum dado.

Requisição 


<soapenv:Envelope xmlns:soapenv="http://schemas.xmlsoap.org/soap/envelope/" xmlns:v300="http://www.cnj.jus.br/mni/v300/" xmlns:tip="http://www.cnj.jus.br/mni/v300/tipos-servico-intercomunicacao">
  <soapenv:Header/>
  <soapenv:Body>
    <v300:requisicaoConsultarCompetencias>
      <tip:codigoLocalidade>CODIGO LOCALIDADE</tip:codigoLocalidade>
    </v300:requisicaoConsultarCompetencias>
  </soapenv:Body>
</soapenv:Envelope>
